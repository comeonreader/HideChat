# 测试与验收标准 v1.2

适用范围：

* 当前版本范围：邮箱登录、幸运数字伪装入口、联系人、1V1 会话、文本 / 图片 / 文件消息、认证后直接进入聊天、本地必要消息缓存

---

## 1. 总体验收原则

必须满足：

* `luckyCode` 允许作为后端校验例外项明文存储
* 会话列表真实内容脱敏
* 搜索结果不返回邮箱
* 图片与通用文件附件链路可用
* 移动端与 PC 端主流程可用
* 历史消息无需 PIN 解锁
* 本地消息缓存恢复不依赖 PIN

---

## 2. 伪装入口模块

### 2.1 运势页展示

测试点：

* 首页默认展示伪装内容
* 手机和 PC 均正常展示
* 文案接口异常时前端有兜底

### 2.2 幸运数字校验

测试点：

* 输入数字后调用后端校验接口
* 命中 `luckyCode` 可进入隐藏入口
* 未命中 `luckyCode` 时仍停留在伪装链路
* 不提示“密码错误”字样

---

## 3. 用户与联系人模块

### 3.1 获取当前用户信息

测试点：

* 已登录用户获取信息成功
* 未登录访问返回 401

### 3.2 搜索用户

测试点：

* 可按昵称搜索
* 可按用户 ID 搜索
* 搜索结果不包含邮箱
* 返回 `matchType`
* 返回 `alreadyAdded`

### 3.3 添加联系人

测试点：

* 添加有效联系人成功
* 重复添加幂等
* 不能添加自己

### 3.4 联系人列表

测试点：

* 正确展示已添加联系人
* 按最近消息时间排序

### 3.5 最近添加列表

测试点：

* 最近添加联系人可见
* 按 `created_at desc` 排序

---

## 4. 会话模块

### 4.1 创建或获取 1V1 会话

测试点：

* 与联系人发起聊天时创建会话
* 已存在会话返回原会话
* A-B 和 B-A 不重复创建

### 4.2 会话列表

测试点：

* 能正确展示聊过的用户
* 最后消息时间正确
* 未读数正确
* 在线状态摘要正确
* 不返回 PIN 相关字段

### 4.3 会话列表脱敏规则

测试点：

* 文本消息只显示 `[文本消息]`
* 图片消息只显示 `[图片消息]`
* 文件消息只显示 `[文件消息]`
* 不展示真实消息文本
* 不展示真实文件名

### 4.4 进入会话

测试点：

* 点击会话直接进入聊天
* 不出现 PIN 设置 / 输入 / 解锁页

---

## 5. 消息模块

### 5.1 文本消息发送

测试点：

* 在线用户之间发送成功
* ACK 正常返回
* 对端实时收到消息

### 5.2 图片消息发送

测试点：

* 图片上传签名获取成功
* 前端直传成功
* 上传完成回调成功
* 图片消息发送成功

### 5.3 文件消息发送

测试点：

* 非图片文件可上传
* 文件消息发送成功
* 聊天页正确展示文件名、大小
* 可下载

### 5.4 历史消息拉取

测试点：

* 可拉取历史消息
* 游标分页正常
* 会话隔离正确
* 无权限用户不能拉取他人会话

### 5.5 已读回执

测试点：

* 标记已读成功
* 对端收到已读通知
* 重复已读幂等

---

## 6. 本地缓存与刷新恢复

### 6.1 认证成功后进入聊天

测试点：

* 幸运数字通过后进入认证页
* 认证成功后直接进入聊天列表
* 不出现 PIN 页面

### 6.2 刷新恢复

测试点：

* 刷新后保留本地登录态时，可在幸运数字校验后直接恢复聊天
* 不需要再次输入 PIN

### 6.3 本地消息缓存

测试点：

* `message_local` 可恢复最近消息
* 本地缓存恢复失败时可回退历史消息接口
* 本地缓存结构不包含 `pinVerifierHash / pinSalt / pinKdfParams / encryptedConversationKey`

---

## 7. 文件与安全测试

### 7.1 文件安全

测试点：

* 非法文件类型上传拦截
* 超大文件拦截
* 非授权访问文件信息拦截

### 7.2 搜索安全

测试点：

* 用户搜索频控有效
* 搜索结果不返回邮箱

### 7.3 本地存储安全

测试点：

* `luckyCode` 仅允许在后端数据库中作为校验例外项明文存储
* 本地消息缓存不依赖 PIN
* 会话列表不暴露真实预览

---

## 8. 最低通过标准

| 模块 | 标准 | 级别 |
| --- | --- | --- |
| 伪装入口 | 命中 luckyCode 后可进入认证或聊天 | 必须 |
| 认证 | 成功后直接进入聊天 | 必须 |
| 会话 | 点击会话直接进入聊天 | 必须 |
| 本地缓存 | 刷新后无需 PIN 仍可恢复使用 | 必须 |
| 敏感信息存储 | 不保存 PIN 材料，不要求 conversationKey 机制 | 必须 |
